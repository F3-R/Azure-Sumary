Módulo 1: Introducción a los fundamentos de Azure

Tipos de modelos en la nube

Actualmente existen tres modelos que dividen el los servicios de la nube según su implementación: nube pública, nube privada y nube híbrida

Nube pública

Este servicio se ofrece a través de Internet y está disponible para cualquier usuario de la red. Este modelo es propiedad de un proveedor de servicios de la nube de terceros.

Principales diferencias:

  • No existen gastos en equipos (CapEx) para escalar verticalmente.
  • Las aplicaciones pueden configurarse para estar disponibles o no disponibles de manera inmediata.
  • En este modelo sólo se paga por lo que se utiliza.

Nube privada

Este servicio se ofrece para usuarios específicos destinados por la empresa u organización que se encuentra encargada de esta. Físicamente puede ubicarse en el centro de datos del encargado o puede estar hospedada por un proveedor de servicios.

Principales diferencias:

  • Se requiere hadware para el inicio y el mantenimiento.
  • Se tiene el control total de los recursos y la seguridad.
  • La responsabilidad del mantenimiento y las actualizaciones del hardware son de la organización

Nube híbrida

Este modelo tiene la facilidad de compartir datos y aplicaciones entre nubes regidas para este modelo debido a que combina las características del modelo de nube pública y privada.

Principales diferencias

  • Ofrece flexibilidad.
  • Se puede determinan el lugar en el que las aplicaciones serán ejecutadas.
  • Se tiene el control de la seguridad, el cumplimiento o el campo legal
